# opis projektu 

Projekt pokazuje prostą stronę portfolio stworząną w HTML, CSS, JavaScript. Strona zawiera sekcje: Stronę główną, informacje o autorze, zainteresowania oraz formularz. W projekcie zastosowano semantyczne elementy HTMLi poprawną hierarchię nagłówków. Do układu elementów wykorzystano technologię Flexbox. Strona jest responsywna działa na komputerach jak i na urządzeniach mobilnych.
